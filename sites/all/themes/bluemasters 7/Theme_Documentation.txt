Theme DocumentationPlease make sure you have Drupal install on your server. If you need help please refer to this guide https://drupal.org/documentation/install/beginnersInfoSheena’s theme is a single page layout Drupal theme based on the Skeleton and BlueMasters template published by Smashing Magazine.Features•	Responsive layout(7.x-2.x)•	Sheena theme is available at https://github.com/sheenasheen/parallax•	Includes Flexslider as an alternative to a background image•	A total of 22 content areas and 8 sections in the navigationInstallation Install Sheena theme the same way you install any modern drupal theme you can read full documentation on installation hereBasicsHow to change the logoWhile logged into the site as an administrator, in the top menu bar select Appearance-> And Settings for the selected theme. Scroll to Logo image settings and upload a new image Change the FaviconWhile logged into the site as an administrator, in the top menu bar select Appearance-> And Settings for the selected theme. Scroll to Short Icon settings and upload a .ico file Using ShortcodesIn order to use the shortcodes included in the skin (ensure that you have installed the included blocks module and that it is enabled) go to structure-> block -> add block. Inside the text editor paste in and modify the html to fit your needs.  Then select the Text format and ensure that Full HTML is selected  and that you specify the region that you want the block to display in. Shortcodes AvailableServices Icon Hovers:  To Achieve these hovers modify the following shortcode :<div class="hi-icon-wrap hi-icon-effect-6">		<a href="#set-6" class="hi-icon hi-icon-contract">Contact</a> </div>Other Included Icons: Row 1:•	hi-icon hi-icon-support•	hi-icon hi-icon-locked•	hi-icon hi-icon-earth•	hi-icon hi-icon-cog•	hi-icon hi-icon-clockRow 2:•	hi-icon hi-icon-videos•	hi-icon hi-icon-list•	hi-icon hi-icon-refresh•	hi-icon hi-icon-images•	hi-icon hi-icon-mobileRow 3:•	hi-icon hi-icon-screen•	hi-icon hi-icon-pencil•	hi-icon hi-icon-link•	hi-icon hi-icon-mail•	hi-icon hi-icon-locationRow 4:•	hi-icon hi-icon-archive•	hi-icon hi-icon-chat•	hi-icon hi-icon-bookmark•	hi-icon hi-icon-user•	hi-icon hi-icon-contractRow 5:•	hi-icon hi-icon-starPortfolio Image Lomo Vingette EffectBlack: <div class="lomo_black">        <img src="/sites/all/themes/bluemasters%207/img/Wave-1.jpg" alt="" />    </div>  White:   <div class="lomo_white">        <img src="/sites/all/themes/bluemasters%207/img/img09.jpg" alt="" />    </div>Natural Language Form		<form id="nl-form" class="nl-form">					OMG!
