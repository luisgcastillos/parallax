Responsive Classes

L : Responsive: Large desktop and up (1200px)
    L-fsize{n}  n > 8 to 80
    usage: L-fsize14 , L-fsize24, L-fsize48 etc.

M : Responsive: Tablet to desktop ( 768px - 979px )
    M-fsize{n} n > 8 to 80
    usage: M-fsize14 , M-fsize24, M-fsize48 etc.

S : Responsive: Landscape phone to desktop/tablet ( max: 767px )
    S-fsize{n} n > 8 to 80
    usage: S-fsize14 , S-fsize24, S-fsize48 etc.

XS: Responsive:  Up to Landscape phone (max: 480px)
    XS-fsize{n} n > 8 to 80
    usage: XS-fsize14 , XS-fsize24, XS-fsize48 etc.
